** Notas **
Las practicas contenidas en este repositorio fueron creadas para funcionar en un microcontrolador Atmega 8535. 
En la raiz de cada carpeta perteneciente a las practicas se encuentra el codigo fuente "practicaX.c" realizado 
en CodeVision, mientras que dentro de la ruta "~\PracticaX-...\Debug\Exe\" se encuentra el archivo ejecutable 
"practicaX.hex" que se debera cargar al microcontrolador por medio del programador universal.    

** Recomendaciones para crear un proyecto en Atmega **
1) Utilizar CodeVision (https://www.codevision.be/) para crear y compilar los programas en lenguaje C (Versión 
evaluación es gratuito y soporta muchos modelos).

2) Realizar la simulación del circuito en Proteus o software similar (Multisim, etc.) para verificar si el programa 
funciona correctamente en el microcontrolador según lo planteado en el código fuente.

3) Descomprimir "Guía Programador-USBasp.zip" y seguir los pasos de la guia en pdf para instalar los controladores
necesarios para poder cargar los programas al microcontrolador con el programador UsbAsp.

Opcional: Se puede utilizar AVR Studio (actualmente nombrado Microchip Studio for AVR and SAM devices) 
(https://www.microchip.com/en-us/tools-resources/develop/microchip-studio) como una alternativa sencilla y mas 
actualizada al software del paso 3, pero es importante revisar si el modelo del microcontrolador que se 
desea utilizar esta soportado por dicho software.
